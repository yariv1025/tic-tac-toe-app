## How to run the game

###### Clone the projcet
``` bash
git clone git@github.com:yariv1025/tic-tac-toe-front-end.git
```

###### Install dependencies
``` bash
npm install
```

###### Run the game - reload at localhost:8080
``` bash
npm run dev
```

You can browse the apis at <http://localhost:8080/#/>
